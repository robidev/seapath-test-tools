This is a repository of tools to test the seapath image.  

It currently inlcudes 4 tools divided over 2 catagories:  

DPDK tools:  
- an IEC 61850 SMV 9-2 packet publisher
- a simple packet reflector

Kernel tools:  
- an IEC 61850 SMV 9-2 packet publisher
- a simple packet reflector

These tools allow to measure latency, jitter and packet-loss of the seapath project, 
when used in conjunction with a separate measuring system using;  

- MoonGen
- a I350 NIC
- provided seapath specific test-scripts

Connected directly with the seapath system via ethernet.
